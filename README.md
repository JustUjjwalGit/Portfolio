# Ujjwal Singh · Personal Portfolio

A personal portfolio built around a Spline 3D scene — black background, futuristic robot, student developer energy.

Built from the [SplineTest](https://github.com/UjjwalOnGit/SplineTest) experiment.

## Sections

- Hero with interactive Spline 3D robot
- About & Interests
- Tech Stack
- Arch Linux terminal card
- Projects (CubeRanked, Weathermap, Planet News, MCSR, Kashdog, AllJavaCodes)
- Timeline
- Personal / Beyond the Code
- Contact / Footer

## Stack

- HTML / CSS / JavaScript
- [Spline Viewer](https://docs.splinetool.com/) for the 3D scene
- Particles, mouse glow, scroll reveal via vanilla JS

## Running

```bash
# serve locally
npx http-server
# or just open index.html
```
